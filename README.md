## Frame

캐스트(Casts)를 상호 작용이 가능한 앱(interactive app) 형태로 변환할 수 있는 표준.  
텍스트 기반의 캐스트 대신 이미지와 버튼이 포함된 프레임을 게시함으로써 다른 유저들과 더 다영한 상호작용을 기대할 수 있다. 

## Frame 관련 라이브러리

- [Frame.js](https://framesjs.org/)
- [Frog.js](https://frog.fm/getting-started)
